The API used for this Website

The REST Countries API is an application programming interface for gathering data about key features of any given nation, primarily including the descriptive entities of official name, capital, region, subregion, population, timezones, currency, language, regional bloc, native name, and land area. However, it has many more statistics that can be referred to in the process.

How the API is Being Used

This API is being used in the quiz by showing quick facts at the end of every question and in the fun-facts page by displaying at least 10 descriptive bits of information when the user inputs a country name in the search field. The variables being requested include the ones mentioned above (name, capital, region, etc.).
